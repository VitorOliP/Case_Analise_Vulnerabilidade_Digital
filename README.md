# Case Analise Vulnerabilidade Digital

## Sobre o Projeto

Este sistema foi desenvolvido para apoiar equipes de atendimento, crédito e prevenção a fraudes na identificação de **clientes com possível vulnerabilidade digital**, principalmente idosos ou pessoas com baixa familiaridade com aplicativos.

A ferramenta utiliza um modelo de **Machine Learning (Random Forest)** treinado em dados simulados, considerando fatores como:

- idade  
- tempo médio de uso do aplicativo  
- taxa de erros de digitação  
- renda anual  
- região  
- contato com suporte  
- histórico de problemas  
- entre outros indicadores operacionais

---

## Objetivo

Ajudar a diferenciar **dificuldade tecnológica real** de **comportamentos suspeitos ou de risco**, permitindo:

- decisões mais humanas e justas  
- redução de falso-positivos  
- atendimento humanizado quando necessário  
- agilidade no fluxo de clientes com boa autonomia digital  

---

## Como o Sistema Funciona

1. Você informa o perfil do cliente na página **“Análise Cliente”**.  
2. O modelo calcula a probabilidade de vulnerabilidade digital.  
3. A aplicação exibe a recomendação apropriada:  
   - atendimento humanizado  
   - ou fluxo normal
